About TrendSpread

TrendSpread is an app/website that tracks worldwide search trends (keywords/phrases) via Google Trends and creates interesting visualizations with the data: 1) a heatmap showing how each trend spreads between countries over the course of a day, 2) a word cloud of keywords, and 3) a pie chart categorizing each of those keywords. All of the data is automatically updated each day and posted to X at @h45hb4ng_data. This also serves as an archive if you would like to see data for a previous day. It is fully open source (click Source menu item).
